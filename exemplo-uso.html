SUNSEEKER Loading Component

Demonstração do componente de loading com a logo SUNSEEKER

Como usar:

1. Incluir os arquivos:

<link rel="stylesheet" href="sunseeker-loading.css"> <script src="sunseeker-loading.js"></script>

2. Uso básico:

// Mostrar loading simples const loading = showSunseekerLoading(); // Ou criar uma instância personalizada const loading = new SunseekerLoading({ duration: 5000, logoPath: 'caminho/para/logo.png' }); loading.show();

3. Loading automático:

<!-- Adicione este elemento para loading automático --> <div id="sunseeker-loading-auto"></div>

Opções disponíveis:

  • duration: Duração em milissegundos (padrão: 3000)
  • logoPath: Caminho para a logo (padrão: 'logo_nitida.png')
  • autoHide: Esconder automaticamente (padrão: true)
  • fadeOutDuration: Duração do fade out (padrão: 500)
